# Cats Escape
> The game about lazy cats and dump mouse

### Run the app

Run game in dev mode: `npm run dev`
Run Levels editor: `npm run levels`

### Mobile Platforms

Run game using cordova:
```
cd mobile_platforms
cp -R ../public/* www/ && cordova run android --device
```
Or just `npm run run-android`
